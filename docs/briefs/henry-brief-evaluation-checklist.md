# Henry Brief Evaluation Checklist

Use this after reading or using the Pipeline Review Defense Brief. The goal is to test whether the brief changes real sales actions, not whether the document looks polished.

## Core Usefulness

- [ ] Did this brief surface a deal I might have under-prioritized?
- [ ] Did it reveal missing context I had not clearly seen?
- [ ] Did it identify an unresolved objection?
- [ ] Did it challenge a forecast assumption?
- [ ] Did it suggest an action I would actually take this week?
- [ ] Did it help me prepare for pipeline review?

## Deal Truth

- [ ] Did the brief distinguish evidence from hope?
- [ ] Did it make weak forecast assumptions visible?
- [ ] Did it show where decision maker, decision timeline, or procurement path is missing?
- [ ] Did it avoid inventing details?
- [ ] Did the forecast evidence category feel fair?

## Action Quality

- [ ] Was the recommended action specific enough?
- [ ] Was the action tied to a real risk?
- [ ] Was the action something I could do this week?
- [ ] Did any action feel generic or obvious?
- [ ] Did any action feel wrong or too confident?

## Manager Review Readiness

- [ ] Could I use the Pipeline Review Answer in an actual review?
- [ ] Would this help me defend a deal?
- [ ] Would this help me decide to downgrade a deal?
- [ ] Would this help me rescue a deal before it goes silent?
- [ ] Would this help me ask better questions to a rep or customer?

## Reflection Questions

1. Which part felt obvious?
2. Which part felt wrong?
3. Which part changed my thinking?
4. Which deal did this brief make me want to act on first?
5. Which missing context is most painful?
6. Which objection debt is most commercially dangerous?
7. Would I want this brief every week?
8. Would I pay or ask my company to pay for this if automated?
9. What should be removed?
10. What should be strengthened?
11. What data source would make this brief much more accurate?
12. What part would I trust in a pipeline review today?
13. What part would I not trust yet?

## Final Evaluation

Choose one:

- [ ] Strong signal: this should become Memoire's wedge.
- [ ] Useful but needs sharper evidence.
- [ ] Interesting but not action-changing.
- [ ] Not useful enough for weekly sales management.
