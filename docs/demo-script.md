# Memoire Demo Script

## Demo Scenario

Account: Control Union

Contact: Nam

Scenario: The user just called Nam from Control Union. They are reviewing a proposal but are concerned about lead time and local support. The user needs to send an implementation timeline next Tuesday.

Sample capture:

```text
Just called Nam at Control Union. They are reviewing the proposal but are concerned about lead time and local support. I need to send an implementation timeline next Tuesday.
```

## Key Message

One customer interaction becomes memory, action, and reusable sales knowledge.

## Demo Flow

### 1. Quick Capture

Open Today and paste the sample capture.

Say: "Memoire starts where sales work actually happens: right after a customer interaction."

Click Structure Interaction.

### 2. Structured Interaction

Show the editable structured preview.

Expected fields:

- Account: Control Union
- Contact: Nam
- Interaction summary: reviewing proposal, concern about lead time and local support
- Objection: lead time and local support
- Next Action: send implementation timeline
- Due Date: next Tuesday

Say: "The raw note is preserved, but Memoire turns it into usable Sales Memory."

Click Save to Sales Memory.

### 3. Account Living Memory

Open Accounts, then Control Union.

Show:

- Account Snapshot
- Account Narrative
- Memory Health
- Key Memory Points
- Objection Memory Bank
- Timeline
- Open Actions

Say: "This is not a CRM record. It is the current story of the account."

### 4. Journey

Open Journey.

Show the Sales Memory Flow Map and Active Journey card.

Say: "Journey shows whether the loop is complete: capture, structure, memory, opportunity, action, and Ask Memoire."

### 5. Broken Loop / Memory Health

If the next action exists, show that the loop is healthier.

If a next action is missing or overdue, show Broken Loops / Needs Attention.

Say: "Memoire helps find where customer memory is not becoming action."

### 6. Ask Memoire

Open Ask Memoire.

Select Specific Account: Control Union.

Ask:

```text
What should I do next?
```

Then ask:

```text
What objections have appeared?
```

Say: "Ask Memoire is grounded in selected Sales Memory, not a generic chatbot."

### 7. Follow-up Composer

From the account or action, click Draft Follow-up.

Generate a draft using:

- Goal: Address objection
- Tone: Consultative
- Length: Short or Medium

Say: "Memoire does not send email. It helps turn memory into a concrete follow-up the salesperson controls."

## Closing Line

Memoire helps individual sales professionals remember the customer story, protect follow-up momentum, and build reusable sales knowledge over a career.
