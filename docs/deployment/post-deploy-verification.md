# Post-Deploy Verification

Run this checklist after deploying Memoire Pipeline Defense to Vercel.

- [ ] Open deployed URL.
- [ ] Navigate to `/app/pipeline-defense`.
- [ ] Refresh direct route.
- [ ] Confirm Internal MVP warning banner.
- [ ] Create new brief.
- [ ] Edit metadata.
- [ ] Refresh and confirm local persistence in the same browser.
- [ ] Import CSV.
- [ ] Analyze Deal Risks.
- [ ] Check Review Readiness.
- [ ] Generate This Week's Actions.
- [ ] Enter Review Mode.
- [ ] Export Brief.
- [ ] Print / Save PDF.
- [ ] Confirm warning is hidden in print.
- [ ] Confirm no real customer confidential data is entered.
- [ ] Confirm Vercel access protection/private deployment if needed.

## Privacy Confirmation

- [ ] Confirm reviewers understand that Pipeline Defense data is stored in browser `localStorage`.
- [ ] Confirm reviewers understand local data can be lost if browser data is cleared.
- [ ] Confirm the deployment is not shared beyond the approved internal audience.
