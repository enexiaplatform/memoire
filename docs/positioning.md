# Memoire Positioning

## One-Liner

Memoire is a personal Sales Memory System that turns every customer interaction into account memory, next actions, and reusable sales knowledge.

## Core Contrast

Your CRM tracks the deal. Memoire remembers the story.

## What Memoire Is

- Personal Sales Memory System
- Daily Sales Action Engine
- Career Knowledge Vault for B2B sales professionals

## What Memoire Is Not

- CRM clone
- Task manager
- Note-taking app
- Sales dashboard
- Generic AI chatbot
- Forecasting tool

## Target Users

- B2B sales professionals
- Account managers
- Business development managers
- Technical sales
- Consultative sellers
- Solo sales operators

## Core Value

- Remember customer context
- Avoid missed follow-ups
- See broken loops
- Turn interactions into next actions
- Ask account-specific and opportunity-specific questions
- Learn from recurring sales patterns

## Product Loop

Capture -> Structure -> Memory -> Opportunity -> Action -> Ask -> Learning
