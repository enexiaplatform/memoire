# Weekly Operating Reviews

Store completed weekly operating reviews here.

Use `docs/operations/weekly-operating-review-template-2026-06-17.md` as the template, then save each completed review as:

```text
docs/operations/weekly-reviews/YYYY-MM-DD-operating-review.md
```

Do not include confidential customer content, raw exports, private sales notes, account names, opportunity names, deal values, or unapproved attribution. Use participant aliases and sanitized quotes unless the customer explicitly approved attribution.

C3 remains open until at least one completed review is saved here with funnel metrics, support status, production health, log review, AI cost review, release-gate decisions, a named operator, and any accepted risks.
